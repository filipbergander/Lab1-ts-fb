## Laboration 1 - Programmering i TypeScript DT208G

Skapar en applikation för att kunna lägga till kurser som man avklarat.
En startsida med vy över kurser i en tabell. 
En sida för att lägga till kurser genom ett formulär.

Kurserna lagras i localStorage och skrivs ut i tabellen genom DOM.
Det som går att ange för varje kurs är:
- Kurskod (Unik)
- Kursnamn
- Progression (A, B eller C)
- Kursplan (URL-adress till kursplanen)

Funktion för att radera kurs finns inom tabellen och en knapp för att radera alla kurser finns nedanför tabellen.
